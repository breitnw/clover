# clover

A massively customizable graphical MPD client for music collectors and visualization lovers

> 🚧 This project is under construction!

Clover is a tool for making your own music players and visualizers. It's heavily inspired by the music and media players of yore (WinAmp, Windows Media Player, and kin), brought into the modern era with tons of customization features. It's 202X! We have partial window transparency and shaders in SDL3! We can do anything!

## Roadmap
1. Standalone app with imported configuration (w/ partial transparency on supported systems!)
3. Renderer and graphical editor
4. Shader editor with inputs that can respond to sound
5. Export to/import from configuration
6. Examples: vectorheart, 3d plastic/metallic, monogatari, triple baka, clover (gadget with stickers?), minimalist black rectangles, rek421 inspired (no album art, viz only) billboards, etc

### Some planned widgets
- Album art display (can layer FX)
- Texture (can layer FX)

- Button (can link to different functions, including mpd database edits)
- Frequency spectrum (maybe this is just a FX preset though)

### "It would be really awesome" widgets
